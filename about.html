CoinSpider

Es una herramienta diseñada para aprender a utilizarBackbone.js, una libreria Javascript basada en el patrón de diseñoMVP(model-view-presenter).

Me pareció apropiado iniciar un proyecto Backbone.js tomando como casos de uso lascripto monedasy operaciones bursátiles. Durante los últimos meses el valor de las principales cripto monedas han tenido un incremento significativo originando un mayor interés en las operaciones bursátiles por parte de la opinión pública.

De tal manera, esta herramienta basada en HTML y Javascript (tu puedes ejecutarla en tu ordenador) te permite conocer el cotización actual de las más importantes cripto monedas en las principales plataformas bursátiles.

Coinspider soporta de momento las siguientes cripto monedas:

  • BTC / Mt.Gox
  • BTC / Bitstamp
  • BTC / Btc-e
  • BTC / BTCChina
  • BTC / Localbitcoins
  • LTC / Btc-e
  • trabajando en muchas mas

Desarrollo

El proyecto es de código abierto, siente libre de colaborar o aprender de él. CoinSpider está basado en la filosfía de desarrollo Node.js

A continuación te describo como ejecutar la versión de desarrollo en tu ordenador

$ git clone https://github.com/fuerbtc/coinspider.git
$ cd coinspider
$ bower install
$ grunt
$ cd dist/
$ open index.html
